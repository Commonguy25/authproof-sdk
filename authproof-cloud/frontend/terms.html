2. Description of Service

Authproof Cloud is a hosted delegation log and cryptographic authorization service for AI agents. The Service allows developers to store tamper-evident receipts of AI agent authorization events, verify those receipts programmatically, and maintain an independently auditable record of agent actions.

The Service stores cryptographic hashes of authorization data — not the raw content of instructions or actions. Receipts are designed to be verifiable by third parties without requiring trust in the operator.

The Service is provided via API and is intended for use by software developers and organizations building AI agent systems.
